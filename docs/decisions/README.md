# Decision Log

Architectural decisions for NorskCoach.

One file per decision. Format: `YYYY-MM-DD-[slug].md`.
Capture entries from Claude Code sessions via `/md-sync` (capture mode).

The WHAT lives in the code; this log exists for the WHY — so a rejected approach
is never silently reintroduced, and so a future session understands the constraint
that shaped a choice without re-deriving it.
